# 最小可行网络云盘 (Minimum Viable Cloud Drive)

一个支持断点续传的简单网络云盘后端服务，使用 FastAPI 构建。

## 功能特性

- ✅ **分片上传**：支持大文件分片上传，每片 ≤ 4MB
- ✅ **断点续传**：上传中断后可从中断处继续
- ✅ **HTTP Range 下载**：支持标准 HTTP Range 头，实现下载续传
- ✅ **Token 鉴权**：保护上传/下载接口
- ✅ **文件管理**：文件列表、信息查询、目录浏览
- ✅ **简易部署**：单文件启动，依赖少
- ✅ **开放协议**：使用 GNU AGPL v3 许可证
